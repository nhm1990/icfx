# ICFx Sunburst

Radiale Heatmap-Visualisierung für ICF-Ergebnisse (Vue 3 + TypeScript + Vite + Tailwind CSS v4).

## Quickstart

```bash
npm install
npm run dev
```

Dev-Server läuft auf `http://localhost:5174`.

## Scripts

| Befehl | Beschreibung |
|--------|-------------|
| `npm run dev` | Vite Dev-Server (Turbopack) |
| `npm run build` | Production-Build |
| `npm run preview` | Build lokal testen |
